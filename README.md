# Eisenhower Matrix

A premium, glassmorphism-styled productivity web application based on the Eisenhower Matrix method. Organize your tasks by urgency and importance to maximize your efficiency.

## 🌐 Live Demo
[**Launch the Eisenhower Matrix**](https://vibeuna.github.io/eisenhower-matrix/)

## ✨ Features
- **Global Task Bar**: Add tasks quickly with optional Due Dates and Notes.
- **2x2 Matrix**: Four clear quadrants (Do First, Schedule, Delegate, Eliminate).
- **Drag and Drop**: Seamlessly drag tasks between quadrants to reprioritize.
- **Edit Modal**: Frosted-glass popup to update task details.
- **Completed Tasks**: dedicated, expandable section to review, restore, or permanently delete completed tasks.
- **Local Storage**: Your data is automatically saved to your browser so it persists between sessions.
- **Premium UI**: Modern dark mode with responsive glassmorphism aesthetics and animated background gradients.

## 🚀 Getting Started
This application is completely client-side (HTML/CSS/JS). No build steps or server required!

1. Clone the repository:
   ```bash
   git clone https://github.com/vibeuna/eisenhower-matrix.git
   ```
2. Open `index.html` in your web browser.

## 🛠️ Built With
- **HTML5**
- **Vanilla CSS3** (Custom Properties, Grid, Flexbox, Glassmorphism)
